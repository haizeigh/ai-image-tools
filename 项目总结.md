> 更新日期：2026-07-02

## 一、项目概述

基于在线工具市场调研，选择 **AI 图像工具** 作为切入点。核心策略：所有处理在浏览器本地完成，不上传服务器，以隐私保护为差异化卖点。

---

## 二、已实现功能

### 7 个工具

| 工具 | 路由 | 技术实现 |
|------|------|---------|
| **AI 去背景** | `/tools/remove-bg` | @imgly/background-removal (Wasm, ONNX Runtime) |
| **图片压缩** | `/tools/compress` | Canvas API，6 种质量 + 3 种宽度限制 |
| **格式转换** | `/tools/format-convert` | JPG/PNG/WebP 互转 |
| **缩放裁剪** | `/tools/resize` | 自定义尺寸 + 6 种社交媒体预设 |
| **证件照制作** | `/tools/id-photo` | 7 种规格 + 4 种背景色，先 AI 去背景再上色 |
| **AI 图片放大** | `/tools/upscale` | Real-ESRGAN (TF.js)，2x-8x，可选 CPU(Wasm)/GPU |
| **Image Resizer** | `/tools/enlarge` | Canvas 插值，放大 2x-10x 或缩小 10%-75% |
| **OCR 文字识别** | `/tools/ocr` | Tesseract.js，20+ 语言 |

## 四、关键设计决策

| 决策 | 说明 |
|------|------|
| **隐私优先** | 所有处理在浏览器端完成，不上传服务器 |
| **AGPL 合规** | 源码公开在 GitHub，Footer 有版权声明 |
| **Wasm 后端** | AI 放大使用 Wasm 而非纯 JS CPU，速度快 50-100 倍 |
| **GPU 可选** | 用户可切换 CPU(Wasm)/GPU，GPU 失败自动回退 Wasm |
| **多语言** | 19 种语言支持，8 种完整翻译 |
| **静态生成** | 所有页面静态生成，部署简单、加载快 |
