## 一、项目背景

## 三、6 个工具详情

### 1. AI 去背景 (`/tools/remove-bg`)
- **技术**: @imgly/background-removal (ONNX Runtime Web)
- **模型**: 预训练神经网络 (~40MB)，浏览器端 Wasm 推理
- **特点**: 支持人像、商品、动物，输出透明背景 PNG
- **隐私**: 不上传服务器

### 2. 图片压缩 (`/tools/compress`)
- **技术**: Canvas API
- **功能**: 可调压缩质量 (10%-100%)、最大宽度限制
- **特点**: 实时显示压缩前后大小对比、压缩率

### 3. 格式转换 (`/tools/format-convert`)
- **技术**: Canvas API
- **支持格式**: JPG、PNG、WebP、GIF、BMP 互转
- **特点**: 自动禁用当前格式选项

### 4. 缩放裁剪 (`/tools/resize`)
- **技术**: Canvas API
- **功能**: 自定义宽高、保持宽高比
- **预设**: Instagram 方形、Twitter 卡片、YouTube 缩略图、LinkedIn 封面、微信头像、4K 壁纸

### 5. 证件照制作 (`/tools/id-photo`)
- **技术**: Canvas API
- **规格**: 一寸、二寸、小一寸、大一寸、美国签证、欧洲签证、日本签证
- **背景色**: 白色、蓝色、红色、灰色

### 6. AI 图片放大 (`/tools/upscale`)
- **技术**: TensorFlow.js + Real-ESRGAN
- **模型**: 三个级别可选
  - 轻量 (3.8 MB) — 默认，速度快
  - 标准 (12 MB) — 均衡
  - 高质量 (112 MB) — 最佳画质
- **特点**: 模型文件内置在项目中，首次加载后浏览器缓存，可离线使用
- **放大倍数**: 4x 超分辨率

## 五、关键设计决策

1. **纯浏览器端处理** — 所有文件不上传服务器，隐私是核心卖点
2. **模型内置** — ESRGAN 模型文件放在 `public/models/`，用户无需联网下载
3. **先选模型再传图** — 放大页面让用户先选模型再上传，流程清晰
4. **SEO 优化** — layout.tsx 配置了中英文 keywords、Open Graph
5. **响应式设计** — 移动端友好
6. **热门标签** — 首页 4 个核心工具标记为"热门"，引导用户
