## 二、功能风险

### 1. AI 去背景（@imgly/background-removal）

| 风险 | 等级 | 说明 |
|------|:----:|------|
| AGPL 合规 | 🟡 已处理 | 源码已公开在 GitHub，footer 有版权声明和源码链接 |
| 模型版权 | 🟢 无风险 | IMG.LY 提供，AGPL 允许商用（需开源） |
| 识别准确率 | 🟢 低风险 | 用户上传的图片效果因人而异，非承诺性服务 |

### 2. OCR（Tesseract.js）

| 风险 | 等级 | 说明 |
|------|:----:|------|
| 识别准确率 | 🟡 低风险 | 已加使用提示，建议用户使用高清图片 |
| 隐私 | 🟢 无风险 | 全部浏览器端处理 |
| 许可证 | 🟢 无风险 | Apache 2.0 |

### 3. AI 放大（Real-ESRGAN + TensorFlow.js）

| 风险 | 等级 | 说明 |
|------|:----:|------|
| 模型版权 | 🟢 无风险 | MIT 许可证 |
| 浏览器兼容性 | 🟡 低风险 | WebGL 可能失败，已做 CPU/Wasm 回退 |
| 推理速度 | 🟢 低风险 | CPU 模式 0.7s（486×450 图片） |

### 4. 其他工具（压缩、转换、缩放、证件照、Resizer）

| 风险 | 等级 | 说明 |
|------|:----:|------|
| 全部使用 Canvas API | 🟢 无风险 | 纯浏览器 API，无外部依赖 |
| 无数据上传 | 🟢 无风险 | 所有处理在浏览器本地完成 |
